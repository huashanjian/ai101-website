# AI101 课程网站

这是 AI101 面向中学生的人工智能通识课程网站骨架，基于 Docusaurus 3 构建。

## 快速开始

1. 安装依赖：npm install
2. 本地启动：npm run start

## 目录结构

- docs/：课程讲义与项目任务
- blog/：课程动态与更新说明
- src/：自定义组件、全局样式与页面
- static/：静态资源（图片、PDF、字体等）
- .github/workflows/deploy.yml：GitHub Actions 部署流水线
